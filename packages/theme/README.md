# @soliviorx/theme

简约风的 Vitepress 博客主题，更多细节见 https://www.zjutshideshan.cn

## Quick Start

① 初始化项目模板

```sh
npx degit SoliviorX/vitepress-blog/packages/template my-blog-demo
```

② 安装依赖

```sh
pnpm install
```

③ 开发启动

```sh
pnpm dev
```

④ 构建产物

```sh
pnpm build
```
